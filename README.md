# Usage
```bash
mkdir ~/.kucing-clang
tar xf clang.tar.zst -C ~/.kucing-clang
echo 'export PATH=$PATH:/home/$USER/.kucing-clang/bin' >> .bashrc
source ~/.bashrc
clang --version
```
